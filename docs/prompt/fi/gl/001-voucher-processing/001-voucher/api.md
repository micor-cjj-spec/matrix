# 凭证接口提示词

## 目标
根据 `docs/bus/fi/gl/001-voucher-processing/001-voucher/` 下业务文档，输出凭证场景的接口清单、请求响应结构、错误码口径、权限与幂等要求，为前后端联调和测试提供统一接口契约。

## 输入上下文
请结合以下文档：
- `business.md`
- `fields.md`
- `flow.md`
- `rules.md`
- `page.md`
- `api.md`
- `docs/prompt/fi/gl/001-voucher-processing/001-voucher/dictionary.md`

同时结合当前已实现能力：列表查询、详情查询、保存、提交、审核、过账、批量过账、分析能力，以及前端已接入的操作链路。

## 输出要求
1. 输出查询、写操作、状态流转三类接口清单
2. 输出单笔与批量接口的请求/响应结构建议
3. 输出分页、详情、错误码、权限与幂等要求
4. 明确保存、提交、审核、过账、冲销等动作的前置条件
5. 对 BUS 未明确项显式标记“待确认”
