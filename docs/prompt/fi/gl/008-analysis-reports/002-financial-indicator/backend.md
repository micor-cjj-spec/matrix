# 财务指标后端提示词

## 目标
根据 `docs/bus/fi/gl/008-analysis-reports/002-financial-indicator/` 下现有业务文档，输出财务指标场景的占位版后端提示词。

## 输入上下文
请仅结合以下已存在文档：
- `business.md`
- `manifest.json`

并严格遵守当前代码事实：
- 未发现后端 controller / service 实现
- 未发现指标计算或批处理实现
- 当前仅为菜单占位

## 输出要求
1. 明确写明“当前无正式后端实现可对齐”
2. 不虚构 service、controller、mapper、表结构或计算流
3. 输出后续后端落地时必须先明确的内容：指标定义来源、计算公式、取数周期、缓存/预计算策略、权限控制与审计日志
4. 所有未落地内容统一标记为“待实现 / 待确认”
