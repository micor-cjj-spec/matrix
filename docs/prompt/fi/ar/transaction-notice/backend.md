# 往来通知单后端开发提示词

## 目标
为往来通知单补充后端开发提示词，支持生成、查询、发送、确认、导出等功能。

## 输入上下文
- `docs/bus/fi/ar/transaction-notice/*.md`
- 现有往来单、账龄分析、信用预警服务

## 开发要求
1. 支持生成通知单草稿，带出余额、到期、逾期信息
2. 支持状态流转：草稿 -> 已发送 -> 已确认 -> 已归档
3. 提供查询接口，支持条件筛选
4. 提供导出接口，保证导出内容与页面一致

## 输出要求
1. 需要修改或新增的类清单
2. 核心业务逻辑说明
3. 接口及返回数据结构说明
4. 文件路径清单
