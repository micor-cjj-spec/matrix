# 总账-资产负债表与利润表业务设计（初版）

> 模块：GL（总账）
> 主题：资产负债表 + 利润表
> 更新时间：2026-03-28

## 1. 业务目标
优先落地资产负债表和利润表，形成三大报表的第一批可上线能力。

选择先做这两张表的原因：
- 当前项目已具备凭证过账与余额表设计
- 科目主数据已有 `freport_item`、`fpltype`
- 相比现金流量表，这两张表取数更稳定、规则更容易闭环

## 2. 总体设计原则
1. 两张表统一基于余额表取数
2. 不直接从页面手工维护金额
3. 科目与报表项目的对应关系尽量配置化
4. 报表结果必须可追溯到余额表、明细账、凭证
5. 先实现标准版本，再考虑多模板和行业差异

## 3. 资产负债表需求设计
### 3.1 业务定位
反映某一时点企业资产、负债、所有者权益结构。

### 3.2 查询条件
- 组织 / 账套
- 会计期间
- 币种
- 模板
- 是否显示零余额项目

### 3.3 展示字段
- 行次
- 项目名称
- 期末余额
- 年初余额
- 附加说明或异常标识

### 3.4 取数口径
- 资产类、负债类、权益类项目均从余额表读取
- 行项目可按单科目、科目区间、多个项目汇总
- 取数方向遵循科目余额方向与报表展示方向转换规则
- 重分类项目可通过公式或规则实现，不建议在前端拼装

### 3.5 关键规则
1. 资产总计必须等于负债和所有者权益总计
2. 若不平衡，需给出差异值和疑似来源
3. 项目可分为明细行、汇总行、公式行
4. 已停用科目不影响历史期间报表
5. 月结后支持固化报表快照

### 3.6 下钻要求
资产负债表每个可钻取行应支持：
- 报表行 -> 科目余额明细
- 科目余额明细 -> 明细账
- 明细账 -> 凭证

## 4. 利润表需求设计
### 4.1 业务定位
反映某期间企业收入、成本、费用和利润形成过程。

### 4.2 查询条件
- 组织 / 账套
- 开始期间
- 结束期间
- 币种
- 模板

### 4.3 展示字段
- 行次
- 项目名称
- 本期金额
- 本年累计金额
- 附加说明或异常标识

### 4.4 取数口径
- 利润表项目优先按科目映射取数
- `fpltype` 作为第一阶段辅助分类字段
- 本期金额来自本期发生额口径
- 本年累计金额为年初至当前期间累计
- 结转损益相关口径需统一，不允许同一组织出现双口径并存

### 4.5 关键规则
1. 营业收入、营业成本、期间费用、税金及附加、其他收益等分类口径需统一
2. 公式行如“营业利润”“利润总额”“净利润”由规则统一计算
3. 允许同一模板下存在企业个性化明细行，但需保持总口径一致
4. 若发现损益类科目未映射，应在查询结果中提示缺失映射

### 4.6 下钻要求
利润表每个可钻取行应支持：
- 报表行 -> 科目发生额明细
- 科目发生额明细 -> 凭证分录
- 凭证分录 -> 原凭证

## 5. 报表项目与科目映射建议
### 5.1 第一阶段
优先复用现有科目主数据中的字段：
- `freport_item`
- `fpltype`

适合快速起步，但限制是：
- 难表达一科目多口径映射
- 难表达生效时间和复杂条件
- 难支撑未来个性化模板

### 5.2 第二阶段
增加独立映射表，支持：
- 同一科目在不同模板下映射不同项目
- 生效期间控制
- 借贷方向转换
- 取数比例或特殊规则

## 6. 页面与交互建议
### 6.1 前端页面
建议新增或扩展以下页面：
- 资产负债表查询页
- 利润表查询页
- 报表项目维护页
- 科目映射维护页

### 6.2 交互能力
- 查询
- 导出 Excel
- 显示零值行开关
- 平衡校验提示
- 缺失映射提示
- 行下钻抽屉

### 6.3 异常提示
需重点提示：
- 报表不平
- 科目未映射
- 余额表未完成重算
- 期间未关账但数据被修改

## 7. 后端接口建议
建议接口：
- `GET /report/balance-sheet`
- `GET /report/profit-statement`
- `GET /report/balance-sheet/drilldown`
- `GET /report/profit-statement/drilldown`
- `GET /report/items`
- `POST /report/items`
- `PUT /report/items/{id}`
- `GET /report/account-mappings`

## 8. 分期建议
### Phase 1
- 报表项目主数据
- 资产负债表查询接口
- 利润表查询接口
- 前端查询与导出
- 基础下钻

### Phase 2
- 模板版本
- 独立科目映射表
- 平衡校验和异常中心
- 快照固化
