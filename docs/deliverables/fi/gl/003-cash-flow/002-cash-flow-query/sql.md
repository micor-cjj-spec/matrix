# 现金流量查询说明

## 1. 数据来源
- 现金相关凭证分析结果
- 现金流项目与活动分类映射
- 识别来源判定结果

## 2. 核心计算口径
- 按识别来源追踪每张现金相关凭证的分类结果
- 统计流入、流出、净额
- 统计直接标记、规则推断、未知编码、多编码复核、现金划转数量

## 3. 核心过滤条件
- 业务单元
- 期间
- 币种
- 现金流项目
- 活动分类
- 识别方式
- 科目编码关键字
- 关键字

## 4. 性能与优化建议
- 期间过滤应优先限制查询范围
- 识别来源与活动分类过滤需避免重复扫描
- 关键字与科目编码模糊匹配应关注性能成本

## 5. 说明
- 当前页面为追踪分析能力，不扩展为写入类能力
