# 现金流量查询运维说明

## 1. 发布关注点
- 前端 `CashFlowQueryView.vue` 与后端 `/cash-flow/query` 接口保持兼容
- 初始化自动查询、业务单元与现金流项目依赖需保持可用
- 查看凭证跳转参数映射需保持一致

## 2. 监控关注点
- 现金流量查询接口响应耗时
- 现金流量查询接口错误率
- warnings 返回异常增长情况
- 业务单元与现金流项目依赖加载失败率

## 3. 风险与回滚
- 若追踪口径异常，优先回滚到已验证的追踪逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若跳转参数不兼容，先保留追踪查询能力，降级跳转能力

## 4. 说明
- 当前场景不涉及写操作，不需要制单类运维方案
