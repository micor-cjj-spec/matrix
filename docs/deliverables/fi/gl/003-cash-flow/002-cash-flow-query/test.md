# 现金流量查询测试说明

## 1. 核心验证范围
- 初始化自动查询
- 查询与重置
- 业务单元、期间、现金流项目、活动分类、识别方式、科目编码关键字、关键字过滤
- 状态条展示
- warnings 展示
- 追踪记录展示
- 查看凭证跳转

## 2. 关键测试点
1. 页面初始化后能自动查询并展示结果
2. 点击查询后返回的统计字段、records、warnings 正确展示
3. 各种过滤条件能正确作用于结果
4. 直接标记、规则推断、未知编码、多编码复核、现金划转状态条统计正确
5. 查看凭证跳转参数正确
6. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 识别来源口径不一致
- warnings 与统计字段口径不一致
- 查看凭证跳转参数映射不一致
