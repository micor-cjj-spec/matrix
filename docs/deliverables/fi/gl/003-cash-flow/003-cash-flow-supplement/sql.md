# 现金流量补充资料查询说明

## 1. 数据来源
- 现金相关凭证分析结果
- 分类分布统计结果
- 待复核与待补录识别结果

## 2. 核心计算口径
- 生成任务清单 `tasks`
- 生成分类分布 `categories`
- 生成待补录/复核凭证 `pendingVouchers`
- 统计直接标记、规则推断、未知编码、多编码、现金划转等数量
- 统计流入、流出、净额

## 3. 核心过滤条件
- 业务单元
- 期间
- 币种

## 4. 性能与优化建议
- 期间过滤应优先限制查询范围
- 分类分布与待补录凭证识别应避免重复扫描
- 统计字段与明细列表应复用基础追踪数据，降低重复计算

## 5. 说明
- 当前页面为补充资料与整改清单展示能力，不扩展为写入类能力
