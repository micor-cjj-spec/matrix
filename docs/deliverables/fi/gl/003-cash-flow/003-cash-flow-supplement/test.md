# 现金流量补充资料测试说明

## 1. 核心验证范围
- 初始化自动查询
- 查询与重置
- 业务单元、期间过滤
- 汇总卡片与状态条展示
- 检查项表、分类分布表、待补录/复核凭证表展示
- warnings 展示
- 查看凭证跳转

## 2. 关键测试点
1. 页面初始化后能自动查询并展示结果
2. 点击查询后返回的统计字段、tasks、categories、pendingVouchers、warnings 正确展示
3. 各种状态条统计正确
4. 查看凭证跳转参数正确
5. 空结果、异常结果、业务单元依赖加载失败能正确展示

## 3. 风险点
- 任务清单、分类分布、待补录/复核凭证口径不一致
- warnings 与统计字段口径不一致
- 查看凭证跳转参数映射不一致
