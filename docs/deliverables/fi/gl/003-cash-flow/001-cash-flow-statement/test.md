# 现金流量表测试说明

## 1. 核心验证范围
- 初始化自动查询
- 查询场景
- 业务单元、期间、币种、模板、showZero 过滤
- checks 展示
- warnings 展示
- 报表行展示
- 跳转按钮

## 2. 关键测试点
1. 页面初始化后能自动查询并展示结果
2. 点击查询后返回的 rows、checks、warnings、template 信息正确展示
3. 不同模板、币种、showZero 条件能正确作用于结果
4. 未知编码、现金划转、启发式分类等提示能正确展示
5. 跳转到现金流量查询与补充资料的参数正确
6. 空结果、异常结果、业务单元加载失败能正确展示

## 3. 风险点
- 活动分类与净增加额口径不一致
- checks / warnings 展示与后端口径不一致
- 模板或跳转参数映射不一致
