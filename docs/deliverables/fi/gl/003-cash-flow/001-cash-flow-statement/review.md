# 现金流量表评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 经营/投资/筹资活动、净增加额、checks、warnings 口径是否一致
- 自动查询、模板信息、跳转按钮、报表展示是否一致
- 业务单元、模板、币种与 showZero 过滤口径是否一致

- 页面定位为现金流主表查询报表
- 接口为 `GET /cash-flow`
- 页面为 `CashFlowView.vue`
- 数据来源包括已过账总账分录、现金流项目主数据、报表模板和报表项目
