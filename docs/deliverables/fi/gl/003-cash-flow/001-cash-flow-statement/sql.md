# 现金流量表查询说明

## 1. 数据来源
- 已过账总账分录
- 现金流项目主数据
- 报表模板
- 报表项目

## 2. 核心计算口径
- 归集经营、投资、筹资三大活动现金流量
- 计算本期现金净流量
- 计算现金及现金等价物净增加额
- 输出 checks 与 warnings

## 3. 核心过滤条件
- 业务单元
- 期间
- 币种
- 模板ID
- 是否显示零值行

## 4. 性能与优化建议
- 期间过滤应优先限制查询范围
- 模板与项目映射需避免重复查询
- 现金流项目匹配与未知编码识别应关注性能成本

## 5. 说明
- 当前报表为查询分析能力，不扩展为写入类能力
