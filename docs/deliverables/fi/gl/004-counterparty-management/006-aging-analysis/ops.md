# 账龄分析表运维说明

## 1. 发布关注点
- 前端 `CounterpartyAgingAnalysisView.vue` 与后端 `/arap-manage/aging-analysis` 接口保持兼容
- 往来方信用额度主数据依赖需保持可用
- 页面只读分析边界需保持一致

## 2. 监控关注点
- 账龄分析接口响应耗时
- 账龄分析接口错误率
- warnings 返回异常增长情况
- 信用额度主数据缺失率

## 3. 风险与回滚
- 若分析口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若风险状态展示不兼容，先保留余额与分桶展示能力，降级风险标识能力

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
