# 账龄分析表口径说明

## 1. 查询条件口径
- `docTypeRoot`：往来类型
- `asOfDate`：统计日期

## 2. 汇总统计口径
- `counterpartyCount`：往来方数
- `warningCount`：预警数
- `totalOpenAmount`：未核销余额
- `normalCount`：正常往来方数

## 3. 表格结果口径
- `rows`：按往来方展示未核销余额结构
- 包含账龄分桶字段、信用额度、逾期阈值、风险状态等信息
- 账龄区间用于展示余额分布，如 0-30 天、31-60 天、61-90 天、90+ 天等

## 4. 风险口径
- 风险状态基于未核销余额、逾期账龄、信用额度、逾期阈值综合判定
- 可识别超额度风险与逾期风险

## 5. warnings 口径
- warnings 由后端返回
- 用于提示高风险往来方、信用额度缺失或账龄异常等问题

## 6. 待确认项
- 账龄区间边界是否需要独立文档化
- 风险状态分级口径是否需要更细说明
