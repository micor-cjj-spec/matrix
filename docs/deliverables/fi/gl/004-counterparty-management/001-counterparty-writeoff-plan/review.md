# 往来核销方案评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 建议核销金额、剩余金额、warnings 口径是否一致
- 查询方案、表格展示、“去自动核销”跳转是否一致
- 往来类型、往来方、统计日期、仅已审核过滤口径是否一致

## 2. 已对齐项
- 页面定位为自动核销前的方案规划页
- 接口为 `GET /arap-manage/plan`
- 页面为 `CounterpartyPlanView.vue`
- 数据来源为 AR/AP 单据与已落库核销链接

## 3. 待确认项
- 方案排序与匹配优先级是否需要独立文档化
- warnings 的正式触发规则是否需要更细说明

## 4. 后续建议
