# 往来核销方案运维说明

## 1. 发布关注点
- 前端 `CounterpartyPlanView.vue` 与后端 `/arap-manage/plan` 接口保持兼容
- 查询方案依赖的往来方与单据数据需保持可用
- “去自动核销”跳转参数映射需保持一致

## 2. 监控关注点
- 核销方案接口响应耗时
- 核销方案接口错误率
- warnings 返回异常增长情况
- 往来方依赖加载失败率

## 3. 风险与回滚
- 若方案口径异常，优先回滚到已验证的匹配逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若跳转参数不兼容，先保留方案预览能力，降级跳转能力

## 4. 说明
- 当前场景不涉及落库核销，不需要执行类运维方案
