# 往来自动核销落库说明

## 1. 数据来源
- 已审核 AR/AP 单据
- 已落库核销链接
- 自动匹配方案结果

## 2. 核心落库口径
- 执行自动核销后写入 `BizfiFiArapWriteoffLog`
- 执行自动核销后写入 `BizfiFiArapWriteoffLink`
- 记录方案号、日志号、核销金额与匹配明细

## 3. 核心过滤与执行条件
- 往来类型
- 往来方
- 统计日期
- 操作人
- 单据需满足已审核等执行前置条件

## 4. 性能与一致性建议
- 执行过程需保证日志与链接写入的一致性
- 已核销链接与剩余金额计算需避免重复写入
- 批量执行时需关注事务边界与并发安全

## 5. 说明
- 当前页面既有预览，也有真实落库执行能力
- 相关统计字段需以后端执行结果为准
