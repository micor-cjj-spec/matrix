# 往来自动核销运维说明

## 1. 发布关注点
- 前端 `CounterpartyAutoWriteoffView.vue` 与 `/arap-manage/plan`、`/arap-manage/auto-writeoff` 接口保持兼容
- 参数带入、自动预览与执行逻辑需保持一致
- 操作人、往来方等依赖需保持可用

## 2. 监控关注点
- 自动核销执行接口响应耗时
- 自动核销执行接口错误率
- warnings 返回异常增长情况
- 日志与链接落库失败率

## 3. 风险与回滚
- 若执行口径异常，优先回滚到已验证的执行逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若带入参数或自动预览不兼容，先保留手工查询与执行能力

## 4. 说明
- 当前场景包含真实落库执行能力，发布需重点关注事务一致性
