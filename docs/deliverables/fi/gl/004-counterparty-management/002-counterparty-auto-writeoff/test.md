# 往来自动核销测试说明

## 1. 核心验证范围
- 方案页带入条件并自动预览
- 预览方案
- 执行自动核销
- 汇总卡片展示
- 成功提示区展示
- warnings 展示
- 匹配明细展示

## 2. 关键测试点
1. 从方案页带入条件后页面能自动预览方案
2. 点击执行自动核销后返回的 `planCode / logId / message / linkCount / totalAmount` 正确展示
3. 执行后核销日志与核销链接落库结果正确
4. warnings 展示正确
5. 重复执行防护与异常处理正确
6. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 方案预览与实际执行结果不一致
- 落库日志与链接数量口径不一致
- 成功提示区字段映射不一致
