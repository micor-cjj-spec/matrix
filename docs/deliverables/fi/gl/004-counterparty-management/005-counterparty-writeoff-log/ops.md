# 往来核销日志运维说明

## 1. 发布关注点
- 前端 `CounterpartyWriteoffLogView.vue` 与后端 `/arap-manage/writeoff-log` 接口保持兼容
- 日志表与链接表数据需保持可用
- 按方案号下钻与批次/明细联动口径需保持一致

## 2. 监控关注点
- 核销日志接口响应耗时
- 核销日志接口错误率
- warnings 返回异常增长情况
- 按方案号下钻明细失败率

## 3. 风险与回滚
- 若日志口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若按方案号下钻不兼容，先保留批次查询能力，降级明细联动能力

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
