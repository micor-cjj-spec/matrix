# 往来核销日志查询说明

## 1. 数据来源
- `BizfiFiArapWriteoffLog`
- `BizfiFiArapWriteoffLink`
- 往来方维度数据

## 2. 核心计算口径
- 统计批次数
- 统计链接数
- 汇总核销金额
- 生成批次摘要
- 按方案号下钻生成链接明细

## 3. 核心过滤条件
- 往来类型
- 往来方
- 方案号
- 开始日期
- 结束日期

## 4. 性能与优化建议
- 时间区间过滤应优先限制日志范围
- 方案号过滤应支持快速下钻明细
- 日志表与链接表关联应避免重复扫描

## 5. 说明
- 当前页面为审计追踪与结果复核能力，不扩展为执行类能力
