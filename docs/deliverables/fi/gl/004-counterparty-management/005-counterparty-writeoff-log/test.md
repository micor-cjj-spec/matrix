# 往来核销日志测试说明

## 1. 核心验证范围
- 查询场景
- 往来类型、往来方、方案号、开始日期、结束日期过滤
- 汇总卡片展示
- warnings 展示
- 批次表展示
- 明细表展示
- 点击批次查看明细联动

## 2. 关键测试点
1. 点击查询后能正确展示日志结果
2. 不同过滤条件能正确作用于结果
3. 批次数、链接数、核销金额展示正确
4. 点击批次行后能带回 `planCode` 并刷新明细表
5. warnings 展示正确
6. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 批次摘要与明细口径不一致
- 方案号联动逻辑不一致
- 核销金额统计口径不一致
