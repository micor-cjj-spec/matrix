# 往来对账单测试说明

## 1. 核心验证范围
- 查询场景
- 往来类型、往来方、统计日期、仅看未核销过滤
- 汇总卡片展示
- warnings 展示
- 对账明细表展示
- 最近核销批次表展示
- 关联凭证跳转
- 核销日志跳转

## 2. 关键测试点
1. 点击查询后能正确展示对账结果
2. 不同过滤条件能正确作用于结果
3. 原额、已核销金额、未核销金额、核销率展示正确
4. 最近核销批次摘要展示正确
5. 关联凭证跳转参数正确
6. 核销日志跳转方案号参数正确
7. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 已核销与未核销金额口径不一致
- 最近批次摘要与日志真实数据口径不一致
- 跳转参数映射不一致
