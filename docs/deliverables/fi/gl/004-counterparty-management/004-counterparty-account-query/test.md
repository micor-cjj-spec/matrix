# 往来账查询测试说明

## 1. 核心验证范围
- 查询场景
- 往来类型、单据类型、单据状态、往来方、关键字、统计日期、仅看未核销过滤
- 汇总卡片展示
- warnings 展示
- 单据级结果展示
- 关联凭证查看

## 2. 关键测试点
1. 点击查询后能正确展示查询结果
2. 不同过滤条件能正确作用于结果
3. 原额、已核销、未核销、核销状态、账龄展示正确
4. warnings 展示正确
5. 关联凭证查看口径正确
6. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 账龄与核销状态口径不一致
- 已核销与未核销金额口径不一致
- 关联凭证查看口径不一致
