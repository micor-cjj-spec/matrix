# 往来账查询运维说明

## 1. 发布关注点
- 前端 `CounterpartyAccountQueryView.vue` 与后端 `/arap-manage/account-query` 接口保持兼容
- 查询依赖的往来方与单据数据需保持可用
- 关联凭证查看口径需保持一致

## 2. 监控关注点
- 往来账查询接口响应耗时
- 往来账查询接口错误率
- warnings 返回异常增长情况
- 关键字查询与多条件过滤性能波动

## 3. 风险与回滚
- 若查询口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若关联凭证查看不兼容，先保留查询能力，降级查看能力

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
