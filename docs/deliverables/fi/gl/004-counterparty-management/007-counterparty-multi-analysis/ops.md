# 往来多维分析表运维说明

## 1. 发布关注点
- 前端 `CounterpartyMultiAnalysisView.vue` 与后端 `/arap-manage/multi-analysis` 接口保持兼容
- 分组维度依赖与查询数据需保持可用
- 页面只读分析边界需保持一致

## 2. 监控关注点
- 多维分析接口响应耗时
- 多维分析接口错误率
- warnings 返回异常增长情况
- 不同 groupDimension 切换后的性能波动

## 3. 风险与回滚
- 若分析口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若分组维度展示不兼容，先保留基础汇总能力，降级高级维度展示

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
