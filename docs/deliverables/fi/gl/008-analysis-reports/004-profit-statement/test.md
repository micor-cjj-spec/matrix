# 利润表测试说明

## 1. 核心验证范围
- 初始化自动查询与查询场景
- 业务单元、开始期间、结束期间、币种、模板、showZero 过滤
- warnings 与 checks 展示
- 主表展示

## 2. 关键测试点
1. 页面初始化后能自动加载业务单元并正确查询主表
2. 不同查询条件能正确作用于主表结果
3. 本期金额与本年累计金额展示正确
4. warnings 与 checks 展示正确
5. `showZero` 只影响展示，不应影响计算口径
6. 空结果、异常结果展示正确
7. 当前页面未接入项目下钻弹窗，不应误判为已具备下钻能力

## 3. 风险点
- 本期金额与本年累计金额口径不一致
- `showZero` 对计算与展示边界处理不一致
- 页面能力边界描述与实际接入范围不一致
