# 利润表查询说明

## 1. 数据来源边界
- 当前已确认前端存在主表查询接口调用
- 本轮未定位到后端实现文件
- 因此此处仅说明查询口径边界，不展开具体表名与 SQL 细节

## 2. 核心查询口径
- 主表：按业务单元、期间范围、币种、模板汇总利润表项目金额
- 展示本期金额与本年累计金额
- 支持 `showZero` 控制零值行显示

## 3. 校验口径
- `warnings` 用于提示利润表口径或数据异常
- `checks` 用于展示校验项结果
