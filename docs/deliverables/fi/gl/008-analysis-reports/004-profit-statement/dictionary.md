# 利润表口径说明

## 1. 查询条件口径
- `orgId`：业务单元
- `startPeriod`：开始期间
- `endPeriod`：结束期间
- `currency`：币种
- `templateId`：报表模板
- `showZero`：显示零值行

## 2. 页面展示口径
- 告警区：`warnings`
- 校验区：`checks`
- 主表区：利润表项目行

## 3. 金额口径
- `currentAmount`：本期金额
- `yearAmount`：本年累计金额
- 期间范围用于确定本期和累计取数区间
- `showZero` 用于控制零值行显示，不应改变计算口径

## 4. 当前边界
- 前端已接入主表查询接口
- 当前页面未接入项目下钻弹窗
- 当前文档不展开后端内部实现类与表结构细节
