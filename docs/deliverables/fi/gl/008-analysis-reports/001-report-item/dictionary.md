# 报表项目口径说明

## 1. 查询条件口径
- `templateId`：报表模板
- `itemCode`：项目编码
- `itemName`：项目名称

## 2. 列表展示口径
- `templateName`：模板名称
- `itemCode`：项目编码
- `itemName`：项目名称
- `rowNo`：行次
- `level`：层级
- `rowType`：行类型
- `periodMode`：期间模式
- `drillable`：可下钻
- `sortNo`：排序

## 3. 查询与树结构口径
- `list`：分页列表，用于页面主展示
- `tree`：项目树，用于树形结构展示或维护支撑
- `detail`：单条项目详情

## 4. 当前页面边界
- 当前页面已接入查询、重置、刷新与分页展示
- 当前页面未接入新增、编辑、删除入口
- 后端写接口已存在，但当前前端未开放维护入口

## 5. 待确认项
- 行类型、期间模式、可下钻的枚举说明是否需要独立文档化
- 树结构查询是否需要在页面后续版本显式展示
