# 报表项目测试说明

## 1. 核心验证范围
- 初始化自动查询与查询场景
- 报表模板、项目编码、项目名称过滤
- 列表展示
- 分页展示
- 刷新与重置
- 后端已提供但前端未接入的接口边界确认

## 2. 关键测试点
1. 页面初始化后能自动加载模板并正确查询列表
2. 不同过滤条件能正确作用于分页结果
3. 模板、项目编码、项目名称、行次、层级、行类型、期间模式、可下钻、排序展示正确
4. 刷新与重置行为正确
5. 分页总数与分页跳转正确
6. tree/detail/write 接口存在，但当前前端未接入，不应误暴露入口

## 3. 风险点
- 模板与项目树层级口径不一致
- 分页查询与树查询口径不一致
- 前端误接入或误暴露未开放维护入口
