# 报表项目查询说明

## 1. 数据来源
- 报表项目表
- 报表模板表

## 2. 核心查询口径
- 分页查询报表项目列表
- 树查询报表项目层级结构
- 按主键查询报表项目详情
- 支持按模板、项目编码、项目名称过滤

## 3. 维护口径
- 支持新增、编辑、删除基础维护能力
- 写操作需关注模板归属、项目编码唯一性、层级与排序一致性

## 4. 性能与优化建议
- 模板过滤、项目编码过滤建议建立索引
- 树结构查询应复用层级与排序字段，避免重复递归开销
- 列表与树查询可根据场景分离返回，避免一次查询承担双重职责

## 5. 说明
- 当前页面主入口为分页列表展示
- 树查询和写接口当前以后端能力保留为主
