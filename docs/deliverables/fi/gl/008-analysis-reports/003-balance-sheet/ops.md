# 资产负债表运维说明

## 1. 发布关注点
- 前端 `BalanceSheetView.vue` 与 `GET /balance-sheet`、`GET /balance-sheet/drill` 接口保持兼容
- 业务单元、模板与报表查询依赖需保持可用
- 当前文档不展开后端内部实现类与表结构细节

## 2. 监控关注点
- 主表接口响应耗时与错误率
- 下钻接口响应耗时与错误率
- warnings 与 checks 返回异常增长情况
- `showZero` 展示行为异常情况

## 3. 风险与回滚
- 若主表查询异常，优先回滚到已验证的前端查询链路
- 若下钻异常，可先保留主表展示能力，临时关闭下钻入口
- 若平衡校验展示异常，应优先核对接口返回事实，不在前端侧推断修正

## 4. 说明
- 当前场景以“前端已接入、后端内部实现未展开”为边界
- 若后续定位到后端实现文件，应再补充更细的后端运维对象说明
