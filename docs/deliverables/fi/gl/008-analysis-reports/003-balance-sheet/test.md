# 资产负债表测试说明

## 1. 核心验证范围
- 初始化自动查询与查询场景
- 业务单元、期间、币种、报表模板、showZero 过滤
- 元信息条展示
- 汇总卡片展示
- warnings 与 checks 展示
- 主表展示
- 下钻弹窗展示

## 2. 关键测试点
1. 页面初始化后能自动加载业务单元和模板并正确查询主表
2. 不同查询条件能正确作用于主表结果
3. 资产总计、负债和所有者权益、差额、平衡状态展示正确
4. warnings 与 checks 展示正确
5. 点击可下钻项目能够正确打开下钻弹窗并加载明细
6. `showZero` 只影响展示，不应影响计算口径
7. 空结果、异常结果展示正确

## 3. 风险点
- 主表汇总口径与下钻口径不一致
- 平衡校验口径不一致
- `showZero` 对计算与展示边界处理不一致
