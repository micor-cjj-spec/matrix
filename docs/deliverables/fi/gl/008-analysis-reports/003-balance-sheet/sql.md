# 资产负债表查询说明

## 1. 数据来源边界
- 当前已确认前端存在主表与下钻接口调用
- 本轮未定位到后端实现文件
- 因此此处仅说明查询口径边界，不展开具体表名与 SQL 细节

## 2. 核心查询口径
- 主表：按业务单元、期间、币种、模板汇总报表项目余额
- 下钻：按项目维度展开到会计科目明细
- 支持 `showZero` 控制零值行显示

## 3. 校验口径
- 资产总计与负债和所有者权益总计对比用于平衡校验
- `warnings` 用于提示报表口径或数据异常
- `checks` 用于展示校验项结果

- 当前不展开后端内部 SQL、mapper 和表结构名称
- 若后续定位到后端实现文件，应再补充正式 SQL 级说明
