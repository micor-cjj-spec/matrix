# 资产负债表口径说明

## 1. 查询条件口径
- `orgId`：业务单元
- `period`：期间
- `currency`：币种
- `templateId`：报表模板
- `showZero`：显示零值行

## 2. 页面展示口径
- 元信息条：模板、业务单元、期间、币种
- 汇总卡片：资产总计、负债和所有者权益、差额、平衡状态
- 告警区：`warnings`
- 校验区：`checks`
- 主表区：报表项目行
- 下钻弹窗：项目对应会计科目明细

## 3. 平衡与下钻口径
- 资产总计与负债和所有者权益总计用于平衡校验
- `diff` 或差额字段用于展示是否平衡
- 下钻口径应与主表项目口径保持一致
- `showZero` 用于控制主表零值行显示，不应改变计算口径

## 4. 当前边界
- 前端已接入主表和下钻接口
- 当前文档不展开后端内部实现类与表结构细节
