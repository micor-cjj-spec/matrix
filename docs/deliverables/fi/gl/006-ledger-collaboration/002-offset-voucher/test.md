# 对冲凭证测试说明

## 1. 核心验证范围
- 查询场景
- 开始日期、结束日期、配对状态、关键字过滤
- 汇总卡片展示
- warnings 展示
- 配对结果表展示
- 跳转查看原凭证与对冲凭证

## 2. 关键测试点
1. 点击查询后能正确展示配对结果
2. 不同过滤条件能正确作用于结果
3. 已配对、孤儿记录、原凭证金额、对冲金额展示正确
4. 金额差异与配对状态展示正确
5. warnings 展示正确
6. 跳转查看原凭证与对冲凭证参数正确
7. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 配对状态判断不一致
- 金额差异口径不一致
- 跳转参数映射不一致
