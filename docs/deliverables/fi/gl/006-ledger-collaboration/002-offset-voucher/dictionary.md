# 对冲凭证口径说明

## 1. 查询条件口径
- `startDate`：开始日期
- `endDate`：结束日期
- `matchStatus`：配对状态
- `keyword`：关键字

## 2. 汇总统计口径
- `pairCount`：已配对数量
- `orphanCount`：孤儿记录数量
- `originalAmount`：原凭证金额
- `reverseAmount`：对冲金额

## 3. 配对结果口径
- `rows`：展示原凭证与对冲凭证的配对关系
- 支持识别缺原凭证、缺对冲凭证、金额不一致等情况
- 配对状态用于表示链路是否完整

## 4. warnings 与跳转口径
- warnings 由后端返回
- 行内可分别跳转查看原凭证和对冲凭证

## 5. 待确认项
- 配对状态枚举是否需要独立文档化
- 金额差异的容差规则是否需要更细说明
