# 对冲凭证评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 已配对、孤儿记录、原凭证金额、对冲金额、金额差异、warnings 口径是否一致
- 查询、表格展示、跳转查看原凭证/对冲凭证是否一致
- 开始日期、结束日期、配对状态、关键字过滤口径是否一致

## 2. 已对齐项
- 页面定位为按冲销链路展示原凭证与对冲凭证配对关系的页面
- 接口为 `GET /ledger-collaboration/offset-vouchers`
- 页面为 `OffsetVoucherView.vue`
- 数据来源为凭证表中的冲销编号与备注关联信息

## 3. 待确认项
- 配对状态枚举是否需要独立文档化
- 金额差异容差规则是否需要更细说明
