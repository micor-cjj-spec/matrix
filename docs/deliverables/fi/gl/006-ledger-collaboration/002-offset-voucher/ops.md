# 对冲凭证运维说明

## 1. 发布关注点
- 前端 `OffsetVoucherView.vue` 与后端 `/ledger-collaboration/offset-vouchers` 接口保持兼容
- 凭证链路与冲销编号/备注关联依赖需保持可用
- 跳转查看原凭证与对冲凭证参数映射需保持一致

## 2. 监控关注点
- 对冲凭证接口响应耗时
- 对冲凭证接口错误率
- warnings 返回异常增长情况
- 孤儿记录与金额差异统计异常波动

## 3. 风险与回滚
- 若配对逻辑或金额差异口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若跳转不兼容，先保留配对查询能力，降级外部联动能力

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
