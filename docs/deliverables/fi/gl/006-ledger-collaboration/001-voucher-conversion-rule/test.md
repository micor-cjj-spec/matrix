# 凭证折算规则测试说明

## 1. 核心验证范围
- 查询场景
- 往来类型、关键字过滤
- 汇总卡片展示
- warnings 展示
- 规则清单展示
- 跳转到应收/应付业务单据页面

## 2. 关键测试点
1. 点击查询后能正确展示规则结果
2. 不同过滤条件能正确作用于结果
3. 规则数、已审核单据、已生成凭证、待生成展示正确
4. 覆盖率与最近单据日期展示正确
5. warnings 展示正确
6. 跳转到应收/应付业务单据页面参数正确
7. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 覆盖率与待生成量口径不一致
- 规则映射与实际单据生成关系口径不一致
- 跳转参数映射不一致
