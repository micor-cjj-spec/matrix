# 凭证折算规则运维说明

## 1. 发布关注点
- 前端 `VoucherRuleView.vue` 与后端 `/ledger-collaboration/voucher-rules` 接口保持兼容
- 规则配置与单据生成统计依赖需保持可用
- 跳转到应收/应付业务单据页面参数映射需保持一致

## 2. 监控关注点
- 凭证折算规则接口响应耗时
- 凭证折算规则接口错误率
- warnings 返回异常增长情况
- 覆盖率与待生成量统计异常波动

## 3. 风险与回滚
- 若规则展示或覆盖率口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若跳转不兼容，先保留规则查询能力，降级外部联动能力

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
