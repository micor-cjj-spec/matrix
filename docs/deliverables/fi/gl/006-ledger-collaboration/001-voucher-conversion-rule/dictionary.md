# 凭证折算规则口径说明

## 1. 查询条件口径
- `docTypeRoot`：往来类型
- `keyword`：关键字

## 2. 汇总统计口径
- `ruleCount`：规则数
- `auditedCount`：已审核单据数
- `generatedCount`：已生成凭证数
- `pendingCount`：待生成单据数

## 3. 规则清单口径
- `rows`：展示单据类型、借贷科目、规则映射、覆盖率、最近单据日期等信息
- 覆盖率用于体现规则对实际业务单据的覆盖情况
- 最近单据日期用于反映规则最近一次被业务命中的时间

## 4. warnings 与跳转口径
- warnings 由后端返回
- 行内可跳转到应收或应付业务单据页面查看具体单据

## 5. 待确认项
- 覆盖率的精确计算公式是否需要独立文档化
- 待生成量是否需区分不同异常原因
