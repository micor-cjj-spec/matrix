# 凭证折算规则评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 规则数、已审核单据、已生成凭证、待生成、覆盖率、最近单据日期、warnings 口径是否一致
- 查询、表格展示、跳转到应收/应付业务单据页面是否一致
- 往来类型、关键字过滤口径是否一致

## 2. 已对齐项
- 页面定位为展示系统当前内置的往来单据转凭证映射规则
- 接口为 `GET /ledger-collaboration/voucher-rules`
- 页面为 `VoucherRuleView.vue`
- 数据来源为内置 `RuleSpec` 与 AR/AP 单据实际生成情况

## 3. 待确认项
- 覆盖率的精确计算公式是否需要独立文档化
- 待生成量是否需要区分不同异常原因

## 4. 后续建议
- 若后续增加规则导出或调整入口，应同步补接口与测试说明
- 若引入预计算规则覆盖率摘要，应同步更新业务口径与 SQL 说明
