# 科目余额对照测试说明

## 1. 核心验证范围
- 初始化自动查询与查询场景
- 开始日期、结束日期、科目编码、仅看差异过滤
- 汇总卡片展示
- warnings 展示
- 科目对照结果表展示
- 跳转到科目余额表页面

## 2. 关键测试点
1. 页面初始化后能自动查询并正确展示对照结果
2. 不同过滤条件能正确作用于结果
3. 科目数、差异科目数、凭证借贷合计、总账借贷合计展示正确
4. 借贷差值与余额差额展示正确
5. warnings 展示正确
6. 跳转到科目余额表页面参数正确
7. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 借贷差值与余额差额口径不一致
- 凭证分录口径与总账分录口径不一致
- 跳转参数映射不一致
