# 科目余额对照评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 科目数、差异科目数、凭证借贷合计、总账借贷合计、借贷差值、余额差额、warnings 口径是否一致
- 初始化自动查询、查询、表格展示、跳转到科目余额表页面是否一致
- 开始日期、结束日期、科目编码、仅看差异过滤口径是否一致

## 2. 已对齐项
- 页面定位为对比凭证分录口径与总账分录口径科目差异的页面
- 接口为 `GET /ledger-collaboration/subject-balance-compare`
- 页面为 `SubjectCompareView.vue`
- 数据来源为凭证分录汇总口径与总账分录汇总口径

## 3. 待确认项

- 差异口径是否需要区分期初、发生和期末三个层次单独说明
