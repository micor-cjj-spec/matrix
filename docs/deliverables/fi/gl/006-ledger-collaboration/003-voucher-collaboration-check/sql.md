# 凭证协同检查查询说明

## 1. 数据来源
- 凭证表
- 凭证分录表
- 总账分录表

## 2. 核心计算口径
- 扫描凭证链路一致性
- 统计异常条数与异常凭证数
- 统计高风险问题数与健康凭证数
- 识别借贷不平、缺少分录、缺少总账分录、重复凭证号等问题

## 3. 核心过滤条件
- 开始日期
- 结束日期
- 问题类型
- 问题等级
- 凭证状态
- 仅看异常

## 4. 性能与优化建议
- 日期过滤应优先限制扫描范围
- 凭证表、凭证分录表、总账分录表关联应避免重复扫描
- 问题识别逻辑应复用基础检查结果

## 5. 说明
- 当前页面为检查与分析能力，不扩展为执行类能力
