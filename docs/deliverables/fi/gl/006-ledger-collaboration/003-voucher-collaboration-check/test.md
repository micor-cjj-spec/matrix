# 凭证协同检查测试说明

## 1. 核心验证范围
- 查询场景
- 开始日期、结束日期、问题类型、问题等级、凭证状态、仅看异常过滤
- 汇总卡片展示
- warnings 展示
- 协同检查结果表展示
- 跳转查看凭证

## 2. 关键测试点
1. 点击查询后能正确展示检查结果
2. 不同过滤条件能正确作用于结果
3. 扫描凭证数、异常条数、高风险问题、健康凭证展示正确
4. 问题类型与问题等级展示正确
5. warnings 展示正确
6. 跳转查看凭证参数正确
7. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 问题类型判断不一致
- 高风险问题分级口径不一致
- 跳转参数映射不一致
