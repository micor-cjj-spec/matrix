# 凭证协同检查评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 扫描凭证数、异常条数、高风险问题、健康凭证、问题类型、warnings 口径是否一致
- 查询、表格展示、跳转查看凭证是否一致
- 开始日期、结束日期、问题类型、问题等级、凭证状态、仅看异常过滤口径是否一致

## 2. 已对齐项
- 页面定位为校验凭证表头、凭证分录和总账分录一致性的检查页面
- 接口为 `GET /ledger-collaboration/voucher-check`
- 页面为 `VoucherCollaborationCheckView.vue`
- 数据来源为凭证、凭证分录和总账分录联合检查
