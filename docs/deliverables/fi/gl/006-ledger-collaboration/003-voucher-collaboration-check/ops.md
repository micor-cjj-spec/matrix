# 凭证协同检查运维说明

## 1. 发布关注点
- 前端 `VoucherCollaborationCheckView.vue` 与后端 `/ledger-collaboration/voucher-check` 接口保持兼容
- 凭证、凭证分录、总账分录链路依赖需保持可用
- 跳转查看凭证参数映射需保持一致

## 2. 监控关注点
- 凭证协同检查接口响应耗时
- 凭证协同检查接口错误率
- warnings 返回异常增长情况
- 高风险问题与异常条数统计异常波动

## 3. 风险与回滚
- 若检查逻辑或问题口径异常，优先回滚到已验证的查询逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若跳转不兼容，先保留检查查询能力，降级外部联动能力

## 4. 说明
- 当前场景不涉及执行类落库操作，不需要执行类运维方案
