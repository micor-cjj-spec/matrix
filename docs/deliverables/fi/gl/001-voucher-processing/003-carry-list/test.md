# 结转清单测试说明

## 1. 核心验证范围
- 默认期间查询
- 业务单元切换
- 元信息区展示
- 汇总卡片展示
- 检查清单展示
- 相关凭证展示
- warnings 展示
- 查看凭证跳转

## 2. 关键测试点
1. 初始化时正确加载业务单元列表
2. 默认期间为当前月份
3. 查询与重置后结果正确刷新
4. `tasks`、`relatedVouchers`、`warnings` 展示正确
5. 点击“查看凭证”能带凭证号条件跳转
6. 空结果与异常结果能正确展示

## 3. 风险点
- 业务单元与期间口径不一致
- warnings 展示与后端口径不一致
- 跳转参数与凭证页查询口径不一致
