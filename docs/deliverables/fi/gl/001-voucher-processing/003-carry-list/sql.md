# 结转清单查询说明

## 1. 数据来源
- 组织财务参数
- 会计期间信息
- 默认凭证字配置
- 基础资料健康检查结果
- 当前期间相关凭证数据

## 2. 核心查询与聚合点
- 按业务单元、期间加载期末处理前置检查信息
- 汇总当前期间凭证数量、结转相关凭证数量、凭证金额
- 识别并返回相关凭证列表
- 生成 warnings 与检查项结果

## 3. 性能与索引建议
- 业务单元、期间、凭证号相关字段应支持快速过滤
- 若相关凭证识别逻辑较复杂，应优先关注当前期间查询成本
- warnings 与检查项若依赖多个来源，应避免重复查询

## 4. 说明
- 当前场景是检查与导航查询，不直接落结转凭证
- 若后续增加执行类动作，应单独补充说明
