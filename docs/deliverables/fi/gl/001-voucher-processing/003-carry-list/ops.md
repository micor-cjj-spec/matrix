# 结转清单运维说明

## 1. 发布关注点
- 前端 `CarryForwardListView.vue` 与后端 `/voucher/carry-list` 接口需保持兼容
- 业务单元列表加载能力需保持可用
- warnings、检查项、相关凭证口径调整时需同步验证页面展示

## 2. 监控关注点
- 结转清单接口响应耗时
- 结转清单接口错误率
- 业务单元初始化加载失败率
- warnings 返回异常增长情况
- 查看凭证跳转兼容性问题

## 3. 风险与回滚
- 若检查口径异常，优先回滚到已验证的聚合逻辑
- 若页面展示异常但接口正常，可先回退前端展示变更
- 若跳转不兼容，先保留检查浏览能力，暂停反查跳转

## 4. 说明
- 当前场景不直接执行结转或结账，不需要执行类运维方案
