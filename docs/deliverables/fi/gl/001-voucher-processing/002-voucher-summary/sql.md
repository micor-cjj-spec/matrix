# 凭证汇总表聚合查询说明

## 1. 数据来源
- 聚合来源：`BizfiFiVoucher` 主表
- 当前不读取分录表

## 2. 核心查询条件
- 凭证日期区间
- 凭证状态
- 摘要关键字

## 3. 核心聚合维度
- 按日期分组
- 按状态计数
- 按金额汇总

## 4. 查询与性能建议
- 保证凭证日期字段可支持区间过滤
- 保证状态字段可支持筛选与状态分桶统计
- 摘要关键字查询需要评估模糊匹配性能成本
- 若数据量增大，应优先检查日期 + 状态相关索引

## 5. 说明
- 当前场景是统计查询，不需要单独设计分录级 SQL
- 若后续需要补导出或更复杂聚合，应在不破坏当前主表聚合口径前提下扩展
