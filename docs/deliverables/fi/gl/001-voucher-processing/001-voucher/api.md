# 凭证接口说明

## 1. 主要接口范围
- 列表查询
- 详情查询
- 保存/新增
- 提交
- 审核/驳回
- 过账
- 批量过账
- 冲销
- 删除
- 分录维护

## 2. 关键接口口径
- 列表支持按凭证号、状态、日期范围等条件查询
- 明细返回需包含凭证头与分录行
- 保存接口需允许草稿场景
- 提交、审核、过账需严格执行状态前置校验
- 批量过账需明确结果返回策略
- 冲销需生成与原凭证关联的反向凭证信息

## 3. 错误与权限说明
- 借贷不平衡
- 会计期间不可用
- 当前状态不允许操作
- 权限不足
- 精度或金额格式不符合规则

## 4. 说明
- 当前主流程以凭证处理为核心，不应在接口说明中扩展无确认的外围能力
- 若后续增加导出或更复杂联动，应单独补充说明
