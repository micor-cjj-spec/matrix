# 凭证测试说明

## 1. 核心验证范围
- 列表查询与明细查看
- 凭证明细编辑
- 保存、提交、审核、驳回、过账、冲销、删除
- 批量过账
- 保存失败错误提示

## 2. 关键测试点
1. 借贷平衡校验正确触发
2. 会计期间不合法时禁止提交/过账
3. 状态越权时禁止执行不允许动作
4. 驳回后可编辑边界与当前实现一致
5. 过账后生成总账分录并更新状态
6. 冲销后生成反向凭证并建立关联
7. 批量过账结果与返回口径一致
8. 前端按钮显隐、禁用与状态机一致

## 3. 风险点
- 精度规则与后端口径不一致
- 冲销关联展示与后端生成逻辑不一致
- 批量过账异常路径覆盖不足
