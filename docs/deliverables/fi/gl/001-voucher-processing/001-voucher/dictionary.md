# 凭证数据字典说明

## 1. 核心对象
- 凭证头
- 凭证明细
- 关联对象（组织、账簿、期间、科目、来源单据、冲销凭证）

## 2. 核心状态口径
- 草稿
- 已提交
- 已审核
- 已过账
- 已驳回
- 已冲销 / 已反向关联（按当前实现口径）

## 3. 关键业务口径
- 借贷必须平衡
- 会计期间必须可用
- 已过账凭证不能按草稿口径再次编辑
- 驳回后是否允许编辑需按当前状态边界执行
- 冲销凭证与原凭证需建立关联关系

## 4. 关键字段口径
- 凭证号：业务唯一编号
- 凭证日期：用于期间校验和列表筛选
- 状态：驱动按钮权限与流程动作
- 分录行：记录借贷方向、金额、摘要、科目等
- 合计借方 / 合计贷方：用于平衡校验

## 5. 说明
- 数据字典以业务语义统一为主，不替代数据库字段清单
- 若状态命名或精度口径变更，应同步更新 API、测试与评审文档
