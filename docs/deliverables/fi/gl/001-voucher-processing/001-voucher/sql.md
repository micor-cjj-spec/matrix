# 凭证 SQL 说明

## 1. 核心数据范围
- 凭证主表
- 凭证明细表
- 总账分录表（过账生成）
- 冲销关联字段或关系

## 2. 关键设计点
- 凭证头与分录行主从关系清晰
- 借贷平衡校验依赖分录金额与方向
- 过账后需生成总账分录并保持状态一致
- 冲销需保存原凭证与反向凭证关联
- 日期、状态、凭证号查询需有合理索引支撑

## 3. 精度与金额
- 金额字段需与后端精度规则一致
- 舍入规则不得由 SQL 层自行漂移出业务口径

## 4. 迁移与回滚建议
- 状态字段、关联字段、精度字段变更时需先验证历史数据兼容
- 涉及过账或冲销关联结构调整时需优先准备回滚方案
