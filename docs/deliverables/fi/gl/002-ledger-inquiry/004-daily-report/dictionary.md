# 日报表口径说明

## 1. 查询条件口径
- `startDate`：开始日期
- `endDate`：结束日期
- `accountCode`：科目编码

## 2. 汇总卡片口径
- 日记录数
- 期间借方
- 期间贷方
- 凭证数

## 3. 表格行口径
- 日期
- 当日借方
- 当日贷方
- 当日凭证数
- 余额方向
- 余额

## 4. 余额口径
- 以已过账总账分录为基础按日聚合
- 余额为按日滚动计算结果
- 余额方向应与余额数值口径保持一致

## 5. warnings 口径
- warnings 由后端返回
- 用于提示某天借贷波动或凭证量异常等需要关注的信息

## 6. 待确认项
- 凭证数的最终统计口径是否按凭证唯一号还是按其他业务规则汇总
- warnings 的正式触发规则是否需要单独固化
