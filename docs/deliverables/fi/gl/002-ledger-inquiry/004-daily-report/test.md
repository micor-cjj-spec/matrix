# 日报表测试说明

## 1. 核心验证范围
- 初始化自动查询
- 查询与重置
- 科目选项加载
- summary 展示
- warnings 展示
- 日报表表格展示

## 2. 关键测试点
1. 页面初始化后能自动查询并展示结果
2. 点击查询后返回的 summary、records、warnings 正确展示
3. 点击重置后恢复默认日期并重新查询
4. accountCode 过滤条件能正确作用于结果
5. 按日借方、贷方、凭证数、滚动余额展示正确
6. 空结果、异常结果、科目选项加载失败能正确展示

## 3. 风险点
- 日级统计口径不一致
- 滚动余额与余额方向口径不一致
- 初始化自动查询与实际参数映射不一致
