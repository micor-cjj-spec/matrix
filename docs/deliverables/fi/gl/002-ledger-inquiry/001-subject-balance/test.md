# 科目余额表测试说明

## 1. 核心验证范围
- query 回填
- 默认查询
- 查询与重置
- 科目下拉加载
- summary 展示
- warnings 展示
- 余额表记录展示

## 2. 关键测试点
1. 从路由 query 带入参数时页面正确回填并查询
2. 点击查询后返回的 summary、records、warnings 正确展示
3. 点击重置后恢复默认日期区间并重新查询
4. accountCode 过滤条件能正确作用于结果
5. 空结果、异常结果、下拉加载失败能正确展示

## 3. 风险点
- 期初 / 本期借贷 / 期末余额口径不一致
- query 参数与实际请求参数映射不一致
- 科目下拉依赖异常影响查询流程
