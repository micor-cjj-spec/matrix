# 科目余额表口径说明

## 1. 查询条件口径
- `startDate`：开始日期
- `endDate`：结束日期
- `accountCode`：科目编码

## 2. 汇总卡片口径
- 科目数
- 本期借方
- 本期贷方
- 有发生科目

## 3. 表格行口径
- 每行按科目展示
- 包含期初余额、本期借方、本期贷方、期末余额等核心余额字段

## 4. warnings 口径
- warnings 由后端返回
- 用于提示余额异常、查询边界或需要关注的信息

## 5. 待确认项
- 期初余额口径是否严格以查询区间前累计发生推导
- warnings 的最终触发规则是否需要单独固化
