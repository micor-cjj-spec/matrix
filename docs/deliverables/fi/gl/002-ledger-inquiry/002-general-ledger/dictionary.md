# 总分类账口径说明

## 1. 查询条件口径
- `startDate`：开始日期
- `endDate`：结束日期
- `accountCode`：科目编码

## 2. 汇总卡片口径
- 流水行数
- 借方合计
- 贷方合计
- 涉及科目

## 3. 表格行口径
- 科目编码
- 科目名称
- 日期
- 凭证号
- 摘要
- 借方
- 贷方
- 余额方向
- 余额

## 4. 余额口径
- 以已过账总账分录为基础生成凭证层级流水
- 余额为滚动计算结果
- 余额方向应与余额数值口径保持一致

## 5. warnings 口径
- warnings 由后端返回
- 用于提示异常流水或需要关注的信息

## 6. 待确认项
- 余额方向的最终展示规则是否需要更细粒度说明
- warnings 的正式触发规则是否需要单独固化
