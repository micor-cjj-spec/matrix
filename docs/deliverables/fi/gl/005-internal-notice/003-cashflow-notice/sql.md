# 现金流通知单查询与生成说明

## 1. 数据来源
- 现金流补充资料待处理凭证
- 内部通知单表

## 2. 核心计算口径
- 查询当前通知清单
- 生成最新通知单
- 统计通知数、未解决数、已解决数、高优先级数
- 统计问题金额与未解决占比
- 统计新增、更新、自动关闭数量

## 3. 核心过滤条件
- 业务单元
- 期间
- 通知状态
- 紧急程度
- 问题来源
- 币种

## 4. 性能与优化建议
- 生成逻辑应避免对同一问题重复插入
- 查询与生成共享待处理凭证结果时应复用基础数据
- 通知状态、紧急程度、问题来源与期间过滤应有合理索引支撑

## 5. 说明
- 当前页面同时提供查询与生成能力
- 跳转到凭证页和勾稽页面不改变通知本身状态
