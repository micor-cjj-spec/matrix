# 现金流通知单评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 通知状态、紧急程度、问题来源、问题金额、生成结果统计口径是否一致
- 初始化自动查询、查询、生成、成功提示区、跳转到凭证页/勾稽页面是否一致
- 业务单元、期间、通知状态、紧急程度、问题来源过滤口径是否一致

## 2. 已对齐项
- 页面定位为把现金流补充资料中的待复核凭证转化为内部处理单的页面
- 查询接口为 `GET /internal-notice/cashflow`
- 生成接口为 `POST /internal-notice/cashflow/generate`
- 页面为 `CashflowNoticeView.vue`
- 数据来源为现金流补充资料待处理凭证与内部通知单表
