# 往来通知单查询与生成说明

## 1. 数据来源
- 往来账龄分析结果
- 内部通知单表

## 2. 核心计算口径
- 查询当前通知清单
- 生成最新通知单
- 统计通知数、未关闭数、已关闭数、高优先级数
- 统计通知金额与未清金额
- 统计新增、更新、自动关闭数量

## 3. 核心过滤条件
- 往来类型
- 通知状态
- 紧急程度
- 统计日期

## 4. 性能与优化建议
- 生成逻辑应避免对同一通知重复插入
- 查询与生成共享风险扫描结果时应复用基础数据
- 通知状态、紧急程度与日期过滤应有合理索引支撑

## 5. 说明
- 当前页面同时提供查询与生成能力
- 跳转到对账单和勾稽页面不改变通知本身状态
