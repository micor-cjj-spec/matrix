# 往来通知单测试说明

## 1. 核心验证范围
- 查询场景
- 通知状态、紧急程度、统计日期过滤
- 汇总卡片展示
- 成功提示区展示
- warnings 展示
- 通知清单展示
- 生成通知单
- 跳转到对账单与勾稽页面

## 2. 关键测试点
1. 点击查询后能正确展示通知结果
2. 不同过滤条件能正确作用于结果
3. 生成通知单后 `generatedCount / insertedCount / updatedCount / resolvedAutoCount / message` 展示正确
4. 通知状态、紧急程度、通知金额、未清金额展示正确
5. 跳转到对账单与勾稽页面参数正确
6. 空结果、异常结果、依赖加载失败能正确展示

## 3. 风险点
- 生成结果统计口径不一致
- 通知状态与紧急程度口径不一致
- 跳转参数映射不一致
