# 现金流通知单勾稽评审清单

## 1. 一致性检查
- BUS 与 prompt 是否一致
- 仍需处理、已自然解除、快照金额、当前金额、warnings 口径是否一致
- 初始化自动查询、查询、表格展示、跳转到凭证页是否一致
- 业务单元、期间、通知状态、币种过滤口径是否一致

## 2. 已对齐项
- 页面定位为把历史现金流通知与当前最新扫描结果做对照的勾稽页面
- 接口为 `GET /internal-notice/cashflow/reconcile`
- 页面为 `CashflowNoticeCheckView.vue`
- 数据来源为内部通知单表与最新现金流候选结果
