# 财务模块业务文档

本目录用于沉淀 `fi` 域下除总账 `gl` 外的其他业务文档。

当前规划的核心业务域包括：

- `er/`：费用报销
- `ap/`：应付管理
- `ar/`：应收管理
- `fa/`：固定资产
- `tax/`：税务管理
- `fund/`：资金管理
- `shared/`：共享与审核
- `report/`：财务报表与分析

说明：

1. 各业务域下统一采用 `business.md / fields.md / flow.md / rules.md / page.md / api.md / manifest.json` 的结构。
2. 当前阶段优先沉淀核心业务对象，后续可继续按实际代码和业务范围扩展。
3. 若后续存在薪酬、预算、影像等独立子域，也可继续在 `fi/` 下扩展。
