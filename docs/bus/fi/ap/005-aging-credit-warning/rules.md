# 业务规则

## 账龄规则
- 仅统计 `SUBMITTED / AUDITED` 状态且 `docTypeRoot=AP` 的开放单据。
- 账龄分桶固定为：`0-30 / 31-60 / 61-90 / 91+`。
- 金额汇总基于单据金额直接累加。

## 预警规则
- 当未结金额大于信用额度时，标记 `overLimit=true`。
- 当最大逾期天数大于逾期阈值时，标记 `overdue=true`。
- 只有命中超额度或超逾期的往来方才进入预警列表。

## 拦截规则
- `fblockOnOverLimit=1` 时，命中超额度会阻断提交/审核。
- `fblockOnOverdue=1` 时，命中超逾期会阻断提交/审核。
- 若未开启硬拦截，则页面仍提示风险，但允许继续操作。
