# 流程设计

## 检查流程
1. 前端调用 `GET /ledger-collaboration/voucher-check`。
2. 后端按日期范围查询凭证，并按状态过滤。
3. 读取对应的凭证分录和总账分录。
4. 逐张凭证执行一致性检查，生成 0 到多条问题记录。
5. 若某张凭证无异常且 `onlyIssue=false`，则返回一条健康记录。
6. 前端展示异常清单、统计卡片和告警信息。

## 检查范围
- 表头金额与分录借方合计是否一致
- 分录借贷是否平衡
- 历史状态凭证是否存在总账分录
- 总账分录条数和金额是否与凭证分录一致
- 是否存在重复凭证号
- 未过账凭证是否提前生成总账分录
