# 流程设计

## 对照流程
1. 前端调用 `GET /ledger-collaboration/subject-balance-compare`。
2. 后端先聚合开始日期前的凭证分录与总账分录，得到两套口径的期初净额。
3. 再聚合期间内的借方、贷方和条数。
4. 按科目汇总生成凭证口径与总账口径的期初、本期、期末数据。
5. 计算差异字段，并标记 `MATCH` 或 `DIFF`。
6. 前端展示对照结果、汇总卡片和告警信息。

## 计算逻辑
- 期初净额 = 期初借方累计 - 期初贷方累计
- 期末净额 = 期初净额 + 本期借方 - 本期贷方
- 差异 = 凭证口径 - 总账口径
