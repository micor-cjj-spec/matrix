# 流程设计

## 状态定义
总账分录本身没有独立状态机，它的生命周期依赖来源凭证的过账和冲销动作。fileciteturn223file0L1-L29

## 业务流程
1. 凭证进入 `AUDITED` 状态后允许过账。fileciteturn217file0L88-L96
2. 过账时先删除该凭证已有总账分录，保证幂等。fileciteturn217file0L101-L107
3. 再按凭证明细逐行生成总账分录。fileciteturn217file0L108-L125
4. 凭证冲销时，生成反向凭证和对应反向总账分录。fileciteturn217file0L176-L227

## 特殊说明
- 总账分录不直接人工编辑。
- 总账分录的增删由凭证过账/冲销流程驱动。
