# 流程设计

## 功能阶段
- 汇总分析阶段
- 结转清单阶段

## 汇总分析流程
1. 接收日期、状态、摘要关键字等筛选条件
2. 校验日期范围和状态格式
3. 查询符合条件的凭证
4. 汇总数量、金额和状态分布
5. 返回按日期分组的结构化结果和警告信息

## 结转清单流程
1. 接收组织和期间参数
2. 解析期间，必要时回退到组织财务配置或系统当前月份
3. 读取组织财务配置、会计期间、默认凭证字和基础资料健康检查结果
4. 识别结转类凭证并形成任务清单
5. 返回任务项、相关凭证和警告信息

## 特殊说明
- 结转清单是分析能力，不直接执行结账或结转动作
- 警告信息用于辅助判断，不等价于硬拦截规则
