# 业务规则

## 汇总分析规则
- `startDate` 不能晚于 `endDate`
- 状态条件统一按大写处理
- 摘要关键字只匹配凭证头摘要，不匹配备注和分录摘要
- 当前无匹配数据时应返回警告信息而不是报错

## 结转清单规则
- `period` 若格式非法，应忽略输入并回退到组织参数或系统当前月份
- 若未选择组织，允许返回提示，但不阻断整体分析
- 结转类凭证通过摘要或备注中的关键字识别
- 默认凭证字、会计期间、组织财务配置缺失时应返回待处理任务或警告

## 联动规则
- 结转清单依赖组织财务配置、会计期间、默认凭证字和基础资料检查结果
- 结转清单是辅助检查，不直接修改凭证和期间状态
