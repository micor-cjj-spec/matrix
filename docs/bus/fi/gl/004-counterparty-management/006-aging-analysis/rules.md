# 业务规则

## 查询规则
- `docTypeRoot` 只支持 `AR / AP`。
- `asOfDate` 为空时按当天处理。

## 账龄规则
- 账龄 = 统计日期 - 单据日期。
- 分桶规则：`<=30`、`31-60`、`61-90`、`>90`。
- 仅对仍有未核销余额的源单据参与账龄统计。

## 风险规则
- 超过信用额度时命中 `OVER_LIMIT`。
- 最大账龄超过阈值时命中 `OVERDUE`。
- 两者都命中时，`riskReason` 同时包含两个原因。

## 提示规则
- 后端会提示：账龄分桶基于“原单据金额 - 已落库核销链接金额”的剩余余额计算。
