# 流程设计

## 执行流程
1. 前端先调用核销方案接口做预览。
2. 用户确认后调用 `POST /arap-manage/auto-writeoff`。
3. 后端仅纳入 `AUDITED` 单据，并沿用方案页相同的匹配规则生成结果。
4. 生成方案号 `AR/AP-WO-时间戳`。
5. 写入一条批次日志 `BizfiFiArapWriteoffLog`。
6. 为每条匹配写入一条核销链接 `BizfiFiArapWriteoffLink`。
7. 返回批次信息、链接数、金额和匹配明细。

## 关键说明
- 自动核销使用“最早源单据优先匹配最早结算单据”的贪心规则。
- 本期实现只落日志和链接，不回写单据状态字段。
