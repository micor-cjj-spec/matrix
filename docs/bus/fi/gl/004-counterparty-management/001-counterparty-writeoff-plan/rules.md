# 业务规则

## 查询规则
- `docTypeRoot` 只支持 `AR / AP`。
- `asOfDate` 为空时按当天处理。
- `auditedOnly=true` 时，仅纳入 `AUDITED` 单据；否则纳入 `SUBMITTED / AUDITED`。

## 方案规则
- 仅 `SOURCE / TARGET` 两类可核销角色参与规划。
- 单据未核销余额 = 原单据金额 - 已落库核销链接金额。
- 同往来方内才允许互相匹配。
- 匹配顺序按单据日期、单据号、主键升序。

## 提示规则
- 后端会提示“未核销余额来自原单据金额减已落库核销链接”。
- 方案页只预览，不创建日志、不落库核销链接。
