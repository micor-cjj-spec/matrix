# 基础设置（basic-settings）

对应前端总账分组“基础设置”。

## 子目录
- `001-voucher-type`：凭证类型
- `002-cash-flow-item`：现金流量项目
- `003-report-account-map`：报表科目映射
- `004-dimension-relation-setting`：核算维度关系设置
- `005-dimension-value-range-setting`：核算维度值范围设置
- `006-equity-change-type`：所有者权益变动类型
- `007-impairment-nature`：减值准备性质
- `008-license-plate-item`：车辆牌照号项目
- `009-cost-nature`：成本性质
