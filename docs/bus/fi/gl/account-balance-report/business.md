# 科目余额表业务说明

## 1. 业务名称
科目余额表（Account Balance Report）

## 2. 业务背景
科目余额表用于按期间、科目和组织口径展示会计科目的期初余额、本期借贷发生额和期末余额，是总账月结、对账、复核和经营分析的基础报表之一。

## 3. 业务目标
- 支持财务按期间快速查看科目余额变化
- 支持月结、核对、报表取数前的复核
- 支持按组织、科目区间、本位币口径输出结构化结果

## 4. 单据定位
科目余额表属于 `fi/gl` 域的分析报表对象，不直接产生账务数据，而是基于凭证、总账分录、期间和组织财务参数汇总生成。

## 5. 核心业务内容
- 按组织、期间、科目区间查询余额
- 展示期初余额、借方发生额、贷方发生额、期末余额
- 支持树形汇总和末级科目明细
- 支持零余额过滤和结果导出

## 6. 上下游关系
### 上游
- 总账分录
- 会计期间
- 会计科目
- 组织财务配置

### 下游
- 财务复核
- 月结检查
- 管理报表取数

## 7. 关键控制点
- 查询期间必须明确
- 余额计算口径必须与总账分录和期间口径一致
- 导出结果与页面展示结果必须保持一致
