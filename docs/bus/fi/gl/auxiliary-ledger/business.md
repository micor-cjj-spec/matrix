# 辅助总账业务说明

## 1. 业务名称
辅助总账（Auxiliary Ledger）

## 2. 业务背景
辅助总账用于在总账基础上，按客户、供应商、项目、部门等辅助核算维度查看余额和发生额，解决只看科目余额无法定位到业务对象的问题。

## 3. 业务目标
- 支持按辅助核算维度查询账务结果
- 支持辅助维度汇总和明细穿透
- 支持期间口径下的期初、发生、期末分析

## 4. 单据定位
辅助总账属于 `fi/gl` 域的分析账表对象，是科目余额表的维度扩展。

## 5. 核心业务内容
- 按辅助维度、期间、科目范围查询余额
- 展示期初、借贷发生、期末余额
- 支持汇总视图和明细视图切换
- 支持与辅助明细账联动查看

## 6. 上下游关系
### 上游
- 总账分录
- 科目配置
- 辅助核算维度资料
- 会计期间

### 下游
- 往来核对
- 项目核算分析
- 管理报表

## 7. 关键控制点
- 辅助维度口径必须与科目配置一致
- 汇总口径和明细穿透口径必须一致
- 查询结果应支持导出和复核
