# 组织财务配置业务说明

## 1. 业务名称
组织财务配置（Org Finance Config）

## 2. 业务背景
组织财务配置用于维护组织级财务基础参数。当前代码中包含本位币、当前期间、期间控制模式、默认凭证字、状态等字段，并提供按组织查询、增删改查、分页查询接口。fileciteturn227file0L1-L27 fileciteturn237file0L1-L46

## 3. 业务目标
- 为每个组织维护唯一一套财务基础参数
- 为凭证分析、结转清单、期间控制提供组织级配置来源
- 统一约束本位币、当前期间、默认凭证字等关键参数

## 4. 单据定位
组织财务配置属于 `fi/gl` 域的基础配置对象，不是业务单据，但会直接影响会计期间、凭证分析和结转场景。fileciteturn224file0L107-L211

## 5. 核心业务内容
- 维护组织、本位币、当前期间、期间控制模式、默认凭证字、状态等信息。fileciteturn227file0L12-L27
- 支持按组织查询配置，用于业务运行时快速读取。fileciteturn237file0L20-L22
- 控制每个组织只能存在一条财务配置记录。fileciteturn240file0L83-L90

## 6. 上下游关系
### 上游
- 组织主数据
- 凭证字配置

### 下游
- 凭证分析服务
- 结转清单
- 期间控制

## 7. 关键控制点
- 同一组织只能存在一条财务配置。fileciteturn240file0L83-L90
- 当前期间格式必须为 `yyyy-MM`。fileciteturn240file0L102-L109
- 期间控制模式仅支持 `STRICT / FLEXIBLE`，状态仅支持 `ENABLED / DISABLED`。fileciteturn240file0L111-L125
