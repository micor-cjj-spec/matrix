# 页面说明

## 列表页
- 支持按组织、状态、本位币分页查询。fileciteturn237file0L35-L44
- 展示当前期间、期间控制模式、默认凭证字、状态等关键信息。

## 详情页
- 展示组织、本位币、当前期间、期间控制模式、默认凭证字、状态、备注及时间信息。fileciteturn227file0L12-L27

## 编辑页
- 支持新增和修改组织财务配置。
- `fcurrentPeriod` 应使用 `yyyy-MM` 格式输入。
- `fperiodControlMode` 和 `fstatus` 建议使用枚举下拉。

## 按钮与交互行为
- 列表页：新增、编辑、删除、查看
- 详情页：查看
- 编辑页：保存
