# 业务规则

## 基础规则
- 上传文件不能为空。
- OCR 结果若明显乱码，需自动尝试降级重试。
- 解析后至少要返回可供人工确认的结构化结果。

## 导入规则
- OCR 解析只返回候选结果，不直接生成正式凭证。
- 用户确认后，先保存凭证草稿，再保存凭证明细。
- 若确认请求中缺少凭证头数据，应直接阻断。

## 推断规则
- 日期优先从文本中提取，提取失败回填当天。
- 摘要优先从业务文本抽取，无法识别时回填默认摘要。
- 金额优先提取文本中的最大金额。
- 分录候选优先按模板行抽取，失败时降级为数字行推断。

## 风险规则
- 科目编码、金额、摘要等推断结果必须允许人工修改。
- OCR 仅作为导入辅助，不能替代正式制单校验。
