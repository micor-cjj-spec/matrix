# 凭证OCR导入业务说明

## 1. 业务名称
凭证OCR导入（Voucher OCR Import）

## 2. 业务背景
凭证OCR导入用于将上传的图片或文件先解析成结构化的凭证草稿数据，再由用户确认后落为正式凭证草稿和分录。当前实现采用两步式：先解析，再确认导入。

## 3. 业务目标
- 降低手工录入凭证的成本
- 通过解析结果预览减少直接入库风险
- 支持中文增强识别、乱码重试和基础分录推断

## 4. 单据定位
凭证OCR导入属于 `fi/gl` 域的辅助导入能力，不是独立账务单据，而是凭证草稿的导入入口。

## 5. 核心业务内容
- 上传文件后调用 OCR 解析服务
- 自动抽取凭证日期、摘要、金额和分录候选数据
- 返回原始文本、凭证头草稿、分录候选和提示信息
- 由用户确认后再落成凭证草稿和凭证明细

## 6. 上下游关系
### 上游
- 图片、扫描件、截图、表格等上传文件

### 下游
- 凭证草稿
- 凭证明细

## 7. 关键控制点
- 解析阶段不直接生成正式凭证
- 识别结果需要人工确认和修正
- 解析失败要给出清晰错误信息
