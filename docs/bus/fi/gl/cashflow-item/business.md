# 现金流量项目业务说明

## 1. 业务名称
现金流量项目（Cashflow Item）

## 2. 业务背景
现金流量项目用于辅助识别和归集企业现金流入流出业务口径，常用于凭证分录或后续取数映射，以支撑现金流量表编制和现金流分析。

## 3. 业务目标
- 建立统一的现金流分类口径
- 支撑凭证与现金流量项目的关联
- 为现金流量表和资金分析提供数据基础

## 4. 单据定位
现金流量项目属于总账模块的辅助核算配置数据，通常与科目、凭证分录或报表取数逻辑联动。

## 5. 核心业务内容
- 维护现金流量项目编码、名称、分类和启用状态
- 配置现金流量项目与科目或凭证分录的映射关系
- 支撑手工指定或自动带出的现金流量归属

## 6. 上下游关系
### 上游
- 会计科目
- 凭证分录规则

### 下游
- 凭证分录
- 现金流量表
- 资金分析报表

## 7. 关键控制点
- 现金流量项目分类口径需保持稳定
- 与科目或分录映射时应避免重复归集或遗漏
- 已被历史业务引用的项目停用前应谨慎校验
