# 页面说明

前端页面：`VoucherSummaryView.vue`

## 页面组成
- 查询区：开始日期、结束日期、状态、摘要关键字
- 汇总卡片：凭证总数、合计金额、待处理凭证、异常凭证
- 告警区：展示后端返回的 warnings
- 表格区：按日期展示凭证数量、金额和状态分布

## 页面行为
- 初始化默认查询“本月第一天 ~ 今天”。
- 点击“查询”调用汇总接口。
- 点击“重置”恢复默认日期并重新查询。
- 点击“查看凭证”会新开凭证页，并把当天日期、状态、摘要条件带过去。

## 当前页面不涉及能力
- 不直接修改凭证。
- 不直接读取分录。
- 不支持导出，当前仅浏览与跳转。
