# 页面说明

前端页面：`VoucherView.vue`

## 查询条件
- 凭证号
- 摘要
- 状态
- 开始日期
- 结束日期

## 列表列
- 凭证号
- 日期
- 摘要
- 金额
- 状态
- 关联

## 工具栏按钮
- 新增凭证
- 编辑
- 分录
- 提交
- 审核
- 过账
- 驳回
- 冲销
- 删除
- 导入
- OCR导入
- 打印

## 状态控制
- 草稿/驳回：编辑、分录、提交
- 已提交：审核、驳回
- 已审核：过账
- 已过账：冲销
- 草稿：删除

## 当前页面已接入能力
- 列表查询
- 新增/编辑/删除
- 提交/审核/过账/驳回/冲销
- 分录查询与保存
- OCR导入确认

## 当前页面未接入能力
- 批量过账
- 来源单据查询
- `/voucher/save` 单独草稿保存接口
