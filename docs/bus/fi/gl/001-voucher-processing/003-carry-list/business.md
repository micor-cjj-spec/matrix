# 结转清单业务说明

## 1. 业务名称
结转清单

## 2. 业务定位
结转清单是期末处理前的检查与提示页面，不直接生成结转凭证，而是帮助用户确认当前业务单元、会计期间、默认凭证字、基础资料和本期结转相关凭证是否满足期末处理条件。

## 3. 业务目标
- 在执行结转损益、自动转账、期末调汇、结账前先做准备检查
- 汇总当前期间的关键期末任务状态
- 展示系统识别到的结转相关凭证，便于用户复核

## 4. 数据来源
后端通过 `BizfiFiVoucherAnalysisServiceImpl.carryList` 汇总组织财务参数、会计期间、默认凭证字、基础资料健康检查结果，以及当前期间的凭证数据。
