# 页面说明

前端页面：`CarryForwardListView.vue`

## 页面组成
- 查询区：业务单元、期间
- 元信息区：本位币、当前期间、期间来源、期间状态、默认凭证字、基础资料健康度
- 汇总卡片：本期凭证数、结转相关凭证、本期凭证金额、检查完成度
- 检查清单表：展示期末处理前置检查项
- 相关凭证表：展示本期识别到的结转相关凭证

## 页面行为
- 初始化时先加载业务单元列表。
- 默认期间为当前月份。
- 点击“查询”调用结转清单接口。
- 点击“重置”恢复当前月份并重新查询。
- 点击“查看凭证”会新开凭证页，并带上凭证号查询条件。

## 当前页面能力边界
- 只做检查与导航，不直接生成结转凭证。
- 不直接修改组织财务参数、会计期间和基础资料。
- 不直接执行结账动作。
