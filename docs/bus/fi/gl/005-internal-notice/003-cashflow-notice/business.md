# 现金流通知单业务说明

## 1. 业务名称
现金流通知单

## 2. 业务定位
现金流通知单用于把现金流补充资料中的待复核凭证转化为内部处理单，推动团队补录现金流项目、处理多编码和确认现金划转事项。

## 3. 业务目标
- 从现金流补充资料自动生成内部通知单
- 跟踪问题来源、紧急程度、处理建议和问题金额
- 为凭证查看和通知单勾稽提供入口

## 4. 数据来源
后端通过 `BizfiFiInternalNoticeServiceImpl.generateCashflowNotices/queryCashflowNotices`，基于现金流补充资料待处理凭证与内部通知单表生成结果。
