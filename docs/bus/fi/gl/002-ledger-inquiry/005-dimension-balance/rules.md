# 业务规则

## 查询规则
- 日期格式必须为 `yyyy-MM-dd`。
- 未传 `endDate` 时默认当天；未传 `startDate` 时默认 `endDate` 所在月第一天。
- `startDate` 不能晚于 `endDate`。
- `accountCode` 使用前缀匹配（`likeRight`）。
- `dimensionCode` 精确匹配现金流项目编码。

## 统计规则
- 当前仅支持按现金流项目作为维度查询。
- 未指定维度的分录统一归并为“未指定现金流项目”。
- 维度余额按维度净额换算方向，不依赖科目正常方向。

## 提示规则
- 后端会额外返回维度提示：当前系统仅支持按现金流项目作为维度进行查询，其他辅助维度待扩展。
- 同时返回基础提示：当前尚未按业务单元隔离，且尚未切换为期间余额表。
