# 核算维度余额表业务说明

## 1. 业务名称
核算维度余额表

## 2. 业务定位
核算维度余额表用于按维度查看期初余额、本期借贷发生和期末余额。当前一期实现里，所谓“核算维度”实际只落地为现金流项目维度。

## 3. 业务目标
- 查看现金流项目维度下的余额变化
- 支撑维度分析和现金流核对
- 为后续扩展部门、客户、供应商等辅助维度预留模式

## 4. 数据来源
后端通过 `BizfiFiLedgerQueryServiceImpl.dimensionBalance` 基于已过账总账分录 `BizfiFiGlEntry` 的 `fcashflowItem` 字段聚合生成结果。
