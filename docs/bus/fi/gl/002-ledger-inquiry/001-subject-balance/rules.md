# 业务规则

## 查询规则
- 日期格式必须为 `yyyy-MM-dd`。
- 未传 `endDate` 时默认当天；未传 `startDate` 时默认 `endDate` 所在月第一天。
- `startDate` 不能晚于 `endDate`。
- `accountCode` 使用前缀匹配（`likeRight`），可按科目级次过滤。

## 计算规则
- 当前仅基于已过账总账分录计算。
- 未指定科目编码时，展示当前期间全部已过账科目余额。
- 正常方向来自科目主数据 `BizfiFiAccount.fdirection`。
- 余额方向由净额推导，不直接读取科目方向。

## 提示规则
后端当前固定返回两类基础提示：
- 当前账表查询直接基于已过账总账分录计算，尚未切换到期间余额表。
- 当前总账分录尚未记录业务单元维度，结果暂未按业务单元隔离。
