# 业务规则

## 查询规则
- 日期格式必须为 `yyyy-MM-dd`。
- 未传 `endDate` 时默认当天；未传 `startDate` 时默认 `endDate` 所在月第一天。
- `startDate` 不能晚于 `endDate`。
- `accountCode` 使用前缀匹配（`likeRight`）。
- `dimensionCode` 精确匹配现金流项目编码。

## 聚合规则
- 当前仅支持按现金流项目 + 科目组合进行辅助总账查询。
- 同一组合下，同一日期、同一凭证号、同一摘要的分录会被合并。
- 每个组合的余额独立滚动计算。

## 提示规则
- 未指定科目编码时，后端会提示辅助总账会按现金流项目和科目组合展示全部流水。
- 同时返回维度提示和基础提示。
