# 业务规则

## 查询规则
- 日期格式必须为 `yyyy-MM-dd`。
- 未传 `endDate` 时默认当天；未传 `startDate` 时默认 `endDate` 所在月第一天。
- `startDate` 不能晚于 `endDate`。
- `accountCode` 使用前缀匹配（`likeRight`）。
- `dimensionCode` 精确匹配现金流项目编码。

## 展示规则
- 当前仅支持按现金流项目 + 科目组合进行辅助明细查询。
- 每条分录单独展示，不做凭证层级合并。
- 每个组合的余额独立滚动计算。

## 提示规则
- 未指定科目编码时，后端会提示辅助明细账会展示全部科目与现金流项目组合的分录明细。
- 同时返回维度提示和基础提示。
