# 业务规则

## 查询规则
- `period` 使用 `yyyy-MM`。
- `currency` 为空时按前端默认值 `CNY` 处理。
- `templateId` 为空时允许自动匹配模板。
- `showZero=false` 时，页面期望隐藏零值行。

## 下钻规则
- 仅 `drillable=true` 的报表项目允许下钻。
- 下钻请求会携带 `itemId` 或 `itemCode`。
- 下钻结果会展示映射来源，用于识别显式映射与自动识别。

## 代码现状说明
- 前端交互和接口调用已存在。
- 本轮未在仓库中检出后端实现文件，因此规则仅描述前端可见行为和调用口径。
