# 业务规则

## 单据规则
- 往来方类型、往来方编码、通知日期不能为空
- 通知内容中的余额、到期金额、逾期金额口径必须来源于统一往来分析结果
- 草稿状态允许调整通知内容和备注

## 状态规则
- 草稿态允许编辑和删除
- 已发送后不允许随意修改核心通知内容
- 已确认状态仅允许查看和归档
- 已归档状态不可再次编辑

## 发送规则
- 发送前应校验通知对象和通知内容完整性
- 已发送后应保留发送时间、发送渠道或发送说明
- 重复发送应有明确处理口径，如覆盖发送记录或禁止重复发送

## 输出规则
- 页面展示、导出结果、打印结果必须保持一致
- 确认结果需要保留留痕，便于后续对账和追溯
