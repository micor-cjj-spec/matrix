# 页面说明

前端页面：`AccountSubjectView.vue`、`AccountSubjectForm.vue`

## 路由入口
- 列表：`/finance/base-data/account-subject`
- 维护：`/finance/base-data/account-subject/form/:fid?`

## 页面组成
### 列表页
- 工具栏：新建、编辑、删除、刷新
- 查询区：科目编码、科目名称、业务单元、科目类型、损益类型
- 列表区：科目编码、科目名称、业务单元、科目类型、余额方向、上级科目、损益类型、关联报表项目、现金类标记、是否末级
- 删除确认弹窗

### 维护页
- 基本信息：编码、名称、业务单元、全称、类型、方向、上级科目、一级科目
- 报表口径：损益类型、关联报表项目
- 控制信息：分录控制、控制级别、末级科目、允许新增下级、允许手工录入、允许凭证分录使用
- 科目属性：现金、银行、现金等价物、外币、往来通知、数量核算

## 页面行为
- 列表页初始化会加载业务单元、报表模板、报表项目和会计科目引用数据
- 点击科目编码或编辑按钮进入维护页
- 保存成功后返回列表页
