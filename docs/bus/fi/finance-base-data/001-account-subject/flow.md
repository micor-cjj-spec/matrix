# 流程设计

## 列表查询流程
1. 前端在 `/finance/base-data/account-subject` 打开会计科目列表页。
2. 页面先并行加载：会计科目全量、报表项目、报表模板、业务单元。
3. 用户按科目编码、名称、业务单元、科目类型、损益类型调用 `GET /account-subject/list`。
4. 后端按条件过滤并按科目编码升序返回分页结果。
5. 前端展示业务单元、上级科目、报表项目和现金类标记的可读名称。

## 维护流程
1. 点击“新建/编辑”进入 `AccountSubjectForm.vue`。
2. 表单先加载会计科目、报表项目、模板、业务单元等引用数据。
3. 用户维护基本信息、报表口径、控制信息、科目属性。
4. 前端调用 `POST /account-subject` 或 `PUT /account-subject`。
5. 保存成功后返回列表页。
