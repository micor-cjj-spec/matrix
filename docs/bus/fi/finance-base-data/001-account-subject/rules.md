# 业务规则

## 查询规则
- `fcode`、`fname` 为模糊匹配。
- `forg`、`ftype`、`fpltype` 为精确匹配。
- 列表按 `fcode` 升序返回。

## 维护规则
- 表单要求：科目编码、科目名称、业务单元、科目类型、余额方向、分录录入控制必填。
- 上级科目、一级科目、报表项目为可选引用字段。
- 新增和编辑均直接调用主表 CRUD，当前后端未见额外唯一性或层级合法性校验。

## 属性规则
- 现金、银行、现金等价物、外币、数量核算、往来通知等属性都以 `0/1` 标记保存。
- `fisEntry` 控制是否允许凭证分录使用。
- `fisDetail` 控制是否末级科目。
