## 三、统一代码模式
当前基础资料中以下模块均采用相同前端模式：
- 物料
- 客户
- 供应商
- 币种
- 汇率
- 行名行号
- 国家
- 地区
- 计量单位

统一特征：
1. 前端均为真实页面，不是占位页。
2. 页面统一复用 `useSimpleData.js`。
3. 前端均接入一套标准 CRUD + 审核流接口。
4. 状态流统一为：`DRAFT / SUBMITTED / AUDITED / REJECTED`。
5. 页面规则基本一致：
   - 名称必填
   - 编码可选
   - `DRAFT` 可提交
   - `SUBMITTED` 可审核/驳回
   - `AUDITED` 不允许编辑和删除
